plugin for SynWrite.
gives command, in Plugins menu, which exports any lexer (selected in dialog) into zip file.

packed files:
 install.inf
 name.lcf
 name.cuda-lexmap (helper file for CudaText)
 name.acp (static autocomplete)
 plus additional linked lexers, if they are "hidden" in Lexer Lib dialog
